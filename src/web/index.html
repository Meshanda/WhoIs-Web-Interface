About

Le but de ce projet était de créer une application web de type "WHOIS" en se basant sur la technologie JNDI.
La technologie JNDI permet, entre autre, de rechercher des informations sur des noms de domaines.
Chaque nom de domaine a un certain nombre d'enregristrements DNS qu'il est possible d'interroger, cliquez ici pour plus d'infos.
